Privacy Policy

This Privacy Policy explains how CodePluse AI handles information. CodePluse AI is an open‑source project available on GitHub and the Azure Marketplace as an Azure Container Offer.

1. Introduction

CodePluse AI ("we," "our," or "us") does not operate a hosted service that collects user data. Deployments are self‑hosted by users in their own environments (for example, within their Microsoft Azure subscription, on a local machine, or in private infrastructure).

2. Data Collection

We do not collect, store, or process personal information from users of CodePluse AI. Specifically:

  • No registration or login with us is required.
  • We do not embed tracking, analytics, or advertising cookies in the Software.
  • Any code, text, or files you process through the Software remain entirely within your environment.

3. OpenAI API Key & AI Processing

  • The Software can integrate with OpenAI services and requires an OpenAI API key to enable AI features.
  • You provide and store your API key only within your own deployment (e.g., container environment variable, configuration file, or secret store).
  • We do not receive or access your API key, and it is never transmitted to us.
  • All API requests are made directly from your environment to OpenAI. Any data sent to OpenAI is governed by OpenAI's own terms and privacy policy.

Reference: OpenAI Terms of Use • OpenAI Privacy Policy

4. Self‑Hosted & Marketplace Deployments

When you deploy CodePluse AI from the Azure Marketplace or GitHub, the Software runs entirely within your chosen environment. Your data and configurations remain under your control. We have no access to your instances, data, logs, or API usage.

5. Third‑Party Services

Although the Software itself does not collect data, your deployment may involve third‑party infrastructure such as Microsoft Azure and OpenAI. Their handling of data is subject to their own policies. Please review:

  • Microsoft Privacy Statement
  • Microsoft Azure Legal Terms
  • OpenAI Privacy Policy

6. Security

Because CodePluse AI is self‑hosted, security depends on how you deploy and configure it. We recommend safeguarding your OpenAI API key, following Azure and/or server security best practices, and keeping your deployment up‑to‑date.

7. Changes to this Policy

We may update this Privacy Policy to reflect improvements to the Software or documentation. Updates will be posted in the GitHub repository and/or the Azure Marketplace listing along with the effective date.
